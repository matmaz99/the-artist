# Hyperbolic Geometry

## What it looks like
Curved space tesselations—patterns that expand exponentially toward edges, like Escher's Circle Limit prints.

## How it works  
Map Euclidean space to hyperbolic space using Poincaré disk or upper half-plane models. Geodesics become curves, regular tilings become infinite.

## Parameters
- **model**: Poincaré disk, Klein disk, or upper half-plane
- **tessellation**: triangle, square, or heptagon tiles
- **depth**: recursion level

## Minimal p5.js sketch
```javascript
// Simplified Poincaré disk visualization
function setup() {
  createCanvas(400, 400);
  noLoop();
}

function draw() {
  background(255);
  translate(width/2, height/2);
  
  noFill();
  stroke(0);
  
  // Draw concentric circles (distance bands)
  for (let r = 20; r < 200; r += 20) {
    ellipse(0, 0, r * 2);
  }
  
  // Draw radial lines (geodesics in Poincaré disk)
  for (let a = 0; a < TWO_PI; a += PI/8) {
    let x1 = cos(a) * 20;
    let y1 = sin(a) * 20;
    let x2 = cos(a) * 180;
    let y2 = sin(a) * 180;
    line(x1, y1, x2, y2);
  }
  
  // Boundary circle
  strokeWeight(2);
  ellipse(0, 0, 400);
}

// Real implementation would use hyperbolic transformations
```

## Combinations

**Typical role:** structure — creates curved-space tessellations that expand toward edges

**Works beautifully with:**
- **substitution-tiling**: Hyperbolic tilings — infinite patterns in finite space
- **instanced-rendering**: Render many tiles efficiently on GPU
- **symmetry-mirroring**: Hyperbolic symmetry groups create stunning kaleidoscopic patterns
- **color-palettes**: Color tiles by depth or generation for visual hierarchy

**Creates tension with:**
- flow-fields: Flow wants continuous space; hyperbolic geometry is discretely tiled. Flow within tiles can work.

**Medium fit:** woven-textile, ceramic-glaze

**Explore from here:**
- If you like curved space → also look at signed-distance-fields for organic curved forms
- If you want flatter patterns → try substitution-tiling for Euclidean aperiodic patterns
- To invent something new → try hyperbolic geometry where tile content is generated by reaction-diffusion
